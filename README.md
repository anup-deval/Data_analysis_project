# Medical Appointment No Shows

This project analyzes medical appointment attendance data to understand why patients miss scheduled appointments.

## Project Objective
Identify patterns related to patient no-shows using data cleaning and preprocessing techniques.

## Dataset
- Source: Kaggle Medical Appointment No Shows Dataset  
- Includes patient details such as:
  - Gender  
  - Age  
  - ScheduledDay  
  - AppointmentDay  
  - Scholarship  
  - Hypertension  
  - Diabetes  
  - Alcoholism  
  - Handicap  
  - SMS_received  
  - No_show  

## Work Completed
- Loaded dataset using Pandas  
- Checked dataset shape and structure  
- Reviewed summary statistics  
- Checked missing values  
- Checked duplicate records  
- Removed invalid negative age values  
- Converted date columns to datetime format  
- Renamed inconsistent column names  
- Encoded categorical variables  
- Saved cleaned dataset for further analysis  

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Jupyter Notebook
